# Huffman File Compressor

## Overview

This project implements **Lossless File Compression** using the **Huffman Coding Algorithm** in C++.

The repository contains both:

- Console Version (C++)
- Qt GUI Version

---

## Features

- Huffman Tree Construction
- Frequency Table Generation
- Binary File Compression
- File Decompression
- Compression Ratio Calculation
- User-Friendly Qt GUI

---

## Technologies Used

- C++
- Data Structures & Algorithms
- Huffman Coding
- Qt 6
- CMake
